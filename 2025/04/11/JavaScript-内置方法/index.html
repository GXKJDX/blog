二、JavaScript-内置常用的方法

1. JSON

1.JSON.parse

将 JSON 字符串解析为 JavaScript 对象。

2. JSON.stringify()

将 JavaScript 对象转换为 JSON 字符串。

3. JSON.stringify(obj, replacer, space)

replacer：用于过滤或转换属性
space：格式化缩进，常用于打印美观的 JSON 字符串

4. JSON.parse() （try…catch）

解析非法 JSON 字符串时会抛出错误，建议包裹在 try…catch 中：

  • JSON 中 key 和 string 值都必须是双引号（”），不能用单引号
  • undefined、函数、Symbol 类型无法被序列化

2.String

1. String.prototype.split

将字符串按指定分隔符拆分为数组。

2. String.prototype.replace

替换字符串中的部分内容。

3. String.prototype.trim

去除字符串两端的空白字符。

3.Array

1. Array.prototype.map

对数组中的每个元素执行回调函数，并返回一个新数组。

2. Array.prototype.filter

根据条件过滤数组，返回满足条件的新数组。

3. Array.prototype.reduce

将数组中的元素累积为一个值。

4. Array.prototype.forEach

遍历数组，对每个元素执行回调函数（无返回值）。

5. Array.prototype.includes

检查数组是否包含某个值，返回布尔值。

6. Array.prototype.find

查找数组中满足条件的第一个元素。

7. Array.prototype.some

检查数组中是否有至少一个元素满足条件。

8. Array.prototype.every

检查数组中的所有元素是否都满足条件。

3. Object

1. Object.keys

获取对象的所有键，返回一个数组。

2. Object.values

获取对象的所有值，返回一个数组。

3. Object.entries

获取对象的键值对，返回一个二维数组。

4. Object.assign

将一个或多个对象的属性合并到目标对象。

4. Function.prototype.bind

1. Function.prototype.bind

创建一个新函数，绑定指定的 this 值和参数。

5. Promise

用于处理异步操作。

6. async/await

简化异步操作的语法。

7. Set

用于存储唯一值的集合。

8. Map

用于存储键值对，键可以是任意类型。

9. Date

用于处理日期和时间。

10. Math

提供数学运算的属性和方法。

11. RegExp

用于处理正则表达式。

总结

JavaScript 提供了丰富的内置方法，涵盖了数据处理、异步操作、集合操作等多个方面。掌握这些方法可以极大地提高开发效率！
